성공기준 이해 3.1.3:생소한 단어 (Level AAA)

성공 기준 (SC)

관용어와 전문용어를 포함하여, 특이하거나 제한된 방식으로 사용된 단어나 문구의 정의를 식별할 수 있는 매커니즘을 제공해야 한다.

요약

목표
사용자는 특이한 단어의 의미를 식별하고 학습할 수 있다.
할 일
기술 전문 용어와 특이한 용어에 대한 정의를 제공한다.
중요성
더 많은 사람들, 특히 인지 장애가 있는 사람들이 콘텐츠의 의미를 이해할 수 있다.

의도

특정 장애를 가진 사람들은 직접적이지 않은 단어 사용이나 전문용어 또는 특수한 사용법을 이해하기 어려울 수 있다. 특정 장애를 가진 사람들은 비유적 표현이나 전문적인 용법을 이해하기 어려울 수 있다. 이러한 청중에게 이런 매커니즘을 제공하는 것은 중요하다. 전문가가 아닌 일반 대중을 위한 전문 정보에서도 레벨 A 또는 레벨 AA 준수만을 주장하더라도 이 성공기준을 만족시키는 것이 권장된다.

이점

이 성공 기준은 다음과 같은 인지 장애, 언어 장애, 학습 장애가 있는 사람들에게 도움이 될 수 있다.

  • 단어를 해독하는 데 어려움을 겪는
  • 단어와 문구를 이해하는 것에 어려움을 겪는
  • 이해를 돕기 위한 맥락을 사용하는 데 어려움을 겪는

또한 다음과 같은 시각 장애가 있는 사람들에게도 도움이 될 것이다.

  • 화면 확대기를 사용하여 확대하면 맥락을 잃는

예제

특이한 방식으로 사용되는 단어에 대한 정의가 포함된 텍스트
사전의 "계층"이나 기타 자료의 목록에서 의미를 검색하면 목표한 정의가 표시되고 "계층"의 다른 출처에서 가져온 정의가 표시되지 않도록 구성한다. ("계층"은 올바른 정의를 찾을 가능성이 가장 높은 순서대로 사전 및 기타 참조 자료를 나열한다. 이를 통해 정의를 검색할 때 따를 순서를 제어할 수 있다.)
정의를 포함한 용어집
WCAG 2.0은 "텍스트"라는 단어를 특정한 방식으로 사용한다. 따라서 WCAG 2.0에서 "텍스트"라는 단어가 사용되면 동일한 웹 페이지 내의 용어집에 제공된 "텍스트"의 정의와 연결된다.
페이지 하단에 제공되는 단어의 구체적인 정의
단어에서 해당 정의로의 내부 링크도 페이지 내에 제공된다.

관련 자료

자료는 정보 제공 목적으로만 제공되며 보증을 암시하지 않는다.

참고

아래 목록에 제품이나 공급업체 이름이 포함되어 있다고 해서 웹 콘텐츠 접근성 지침 실무 그룹이나 W3C의 웹 접근성 이니셔티브가 보증하는 것은 아니다. 이 목록은 단지 편의를 위해 제공되며 사용자에게 어떤 리소스를 사용할 수 있는지 알려준다.

  • Freedict.org 웹 사이트에서 다양한 언어에 대한 무료 이중 언어 사전을 이용할 수 있다. 사이트에 명시된 대로 사전의 품질과 크기가 고르지 않다. 2005년 4월 9일에 확인.
  • WorldStar 무료 사전, 번역기 및 검색 엔진 사이트에서는 다양한 언어로 된 무료 온라인 사전 및 검색 엔진에 대한 액세스를 제공한다. 2005년 11월 18일에 확인.
  • 더 많은 사전이 your dictionary, freelang.com(영어, 스페인어, 프랑스어!) 및 기타 여러 곳에서 제공된다.

기법

이 섹션에서 번호가 매겨진 각 항목은 WCAG 실무 그룹이 이 성공 기준을 충족하기에 충분하다고 간주하는 기법 또는 기법의 조합을 나타낸다. 그러나 이러한 특정 기법을 사용할 필요는 없다. 다른 기법 사용에 대한 자세한 내용은 WCAG 성공 기준에 대한 기법 이해, 특히 "기타 기법" 섹션을 참고하라.

충분 기법

아래에서 귀하의 콘텐츠와 일치하는 상황을 선택하라. 각 상황에는 해당 상황에 충분하다고 알려져 있고 문서화된 기법 또는 기법의 조합이 포함된다.

상황 A: If the word or phrase has a unique meaning within the Web page:

  • G101: Providing the definition of a word or phrase used in an unusual or restricted way for the first occurrence of the word or phrase in a Web page using one of the following techniques:

    • G55: Linking to definitions

      • H40: Using description lists

    • G112: Using inline definitions

      • H54: Using the dfn element to identify the defining instance of a word

  • G101: Providing the definition of a word or phrase used in an unusual or restricted way for each occurrence of the word or phrase in a Web page using one of the following techniques:

    • G55: Linking to definitions

      • H40: Using description lists
    • G62: Providing a glossary
    • G70: Providing a function to search an online dictionary

상황 B: If the word or phrase means different things within the same Web page:

  • G101: Providing the definition of a word or phrase used in an unusual or restricted way for each occurrence of the word or phrase in a Web page using one of the following techniques:

    • G55: Linking to definitions

      • H40: Using description lists

    • G112: Using inline definitions

      • H54: Using the dfn element to identify the defining instance of a word
